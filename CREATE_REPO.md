# Создание GitHub репозитория - 30 секунд

## Быстрые действия:

1. **Открой:** https://github.com/new
2. **Название:** `my-markdown-blog` (или любое)
3. **Тип:** Public (для бесплатного Netlify)
4. **НЕ добавляй:** README, .gitignore, license
5. **Нажми:** "Create repository"

## После создания GitHub покажет команды:

```bash
git remote add origin https://github.com/ВАШ_USERNAME/my-markdown-blog.git
git branch -M main
git push -u origin main
```

**Скопируй эти команды и выполни их в терминале!**

Я жду, когда ты создашь репозиторий, чтобы продолжить настройку Netlify.
