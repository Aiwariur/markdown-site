Привет! Это тестовый пост для проверки работы markdown-site фреймворка.

## Что это такое?

Markdown-site — это open-source фреймворк для блогов, который позволяет:

- Писать посты в обычном markdown
- Синхронизировать контент одной командой `npm run sync`
- Получать мгновенные обновления на сайте

## Технологии

Под капотом используются:

- **Convex** — real-time база данных
- **Netlify** — бесплатный хостинг
- **React + TypeScript** — фронтенд
- **Vite** — сборщик
